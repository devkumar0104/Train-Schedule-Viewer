# Train-Schedule-Viewer
A standalone Java desktop application with a fully custom-built GUI (no external UI libraries). Schedules and details for 100+ trains are stored in a structured JSON file, parsed at runtime to deliver fast, searchable timetable lookups.
A Java desktop application for viewing and searching train schedules.

## Features
- Search through schedules of 100+ trains
- Clean GUI built entirely with Java Swing/AWT
- Train data stored and read from a structured JSON file

## How to Run
1. Make sure you have Java installed on your machine
2. Clone the repository
3. Open the project in any Java IDE (IntelliJ, Eclipse, VS Code)
4. Run the main file

## Tech Used
- Java
- Swing/AWT (GUI)
- JSON (data storage)
